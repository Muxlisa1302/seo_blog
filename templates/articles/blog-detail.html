React Hooks: Funksional Komponentlar Kuchi

React Hooks - bu React 16.8 versiyasida taqdim etilgan yangi xususiyat bo'lib, u funksional komponentlarga state va boshqa React xususiyatlaridan foydalanish imkonini beradi. Keling, eng ko'p ishlatiladigan hooklarni ko'rib chiqaylik.

1. useState Hook

useState hook funksional komponentlarga state qo'shish imkonini beradi. Mana oddiy misol:


import React, { useState } from 'react';

function Counter() {
  const [count, setCount] = useState(0);

  return (
    <div>
      <p>Siz {count} marta bosdingiz</p>
      <button onClick={() => setCount(count + 1)}>
        Bosing
      </button>
    </div>
  );
}

2. useEffect Hook

useEffect hook komponentning hayot sikli metodlarini almashtirishga imkon beradi. U komponent render bo'lgandan keyin ishga tushadi:


import React, { useState, useEffect } from 'react';

function Example() {
  const [count, setCount] = useState(0);

  useEffect(() => {
    document.title = `Siz ${count} marta bosdingiz`;
  });

  return (
    <div>
      <p>Siz {count} marta bosdingiz</p>
      <button onClick={() => setCount(count + 1)}>
        Bosing
      </button>
    </div>
  );
}

3. useContext Hook

useContext hook kontekst qiymatlarini olish uchun ishlatiladi:


import React, { useContext } from 'react';

const ThemeContext = React.createContext('light');

function ThemedButton() {
  const theme = useContext(ThemeContext);
  return <button style={{ background: theme }}>Men themed buttonman</button>;
}

React Hooks funksional komponentlarga yangi imkoniyatlar beradi va kodingizni ancha soddalashtirishga yordam beradi. Ularni o'rganish va amalda qo'llash orqali siz yanada samaraliroq React dasturlarini yaratishingiz mumkin.
